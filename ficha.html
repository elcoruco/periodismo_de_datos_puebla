Cambio de placas Gratis en Puebla

Descripción técnica

Aquí el procedimiento:

  1. Me volé un mapa de Puebla en SVG de transparencia presupuestaria.
  2. Busqué las coordenadas de los centros de cambio de placas en Google maps.
  3. Calculé la distancia entre el centro de cada municipio y los doce centros de canje usando la fórmula de Haversine. A cada uno le asigné el "más cercano".
  4. utilicé d3 para colorear el mapa, asignando un verde botella para los municipios con centro de cambio de placas, y un rojo coágulo a los municipios más alejados de uno.
  5. Para completar la información del proyecto, utilicé INEGI fácil para obtener el indicador para automóviles registrados en circulación.
  6. Ya con el indicador, utilicé la librería de PHP de INEGI fácil para obtener los autos en circulación para cada municipio de Puebla.
  7. Por último, agregué el índice de desarrollo humano para cada municipio. Esta información la obtuve de los amigos de intellitrendsMX. Aquí el link a la DB.
  8. Para calcular las distancias y tiempos en carretera, utilcé el Servicio de matriz de distancia, de la API de Google maps.
  9. Aunque se supone es por municipio el asunto, las distancias finales se calcularon en base a la cabecera de cada municipio, por eso se utiliza el nombre de la ciudad y no el del municipio. Sí, algo confuso, pero pues ya se fue así :P.
  10. El proyecto tomó aproximadamente 28 horas de desarrollo, y se armó con PHP y JS.
  11. Como todo lo demás, este proyecto se armó sin dinero. (Pero ya estamos en pláticas con la República del Congo para obtener recursos.)
